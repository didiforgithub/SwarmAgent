# SwarmAgent

现有的Multi-agent框架从用途上可分为两类，一类为Agent赋予不同的角色以实现外部目标，另一类为Agent赋予尽可能类似人类的思维机制探索Agent与Agent之间的社会行为
然而，在社会模拟这一方向的Multi-agent框架中，大部分研究围绕 Agent 与 Agent 之间的关系展开，而并没有考虑Agent组成的群体之间的相互影响。
